# Rust Baseball Game ⚾

Rust と軽量ゲームフレームワーク `macroquad` で開発された、シンプルな2Dミニ野球ゲームです。
ピッチャーが投げる多彩な変化球をバットの芯で捉え、タイミングよく打ち分けてサヨナラ勝ちを目指しましょう！

## 🚀 特徴

- **ダイナミックな打撃ロジック**: スイングのタイミング（早振り・遅振り）によって、レフト方向への引っ張りやライト方向への流し打ちが自然に変化します。
- **バッターのポジショニング**: 投球のコース（内角・外角）に合わせて、バッターを左右に動かして待ち構えることができます。
- **多彩な投球**: 直球だけでなく、不規則な変化を見せる変化球もランダムで投下されます。
- **守備シフト**: 打球が飛んだ方向に野手が自動で追いかけてくる簡易AIを搭載。

## 🕹️ 操作方法

| キー / マウス | 操作内容 |
| :--- | :--- |
| **ENTER キー** | タイトル画面からゲームを開始する |
| **SPACE キー** | ピッチャーに投球させる（Ready状態時） |
| **A / D キー** または **◀ / ▶ キー** | バッターを左右に移動（打席位置の調整） |
| **マウス左クリック** | バットを振る（スイング） |
| **ESC キー** | ゲームセット画面からタイトルに戻る |

## 🛠️ 動作環境・必要なもの

- Rust (MSRV 1.70.0 以上推奨)
- Cargo

## 🏃 起動方法

リポジトリをクローンするかダウンロードした後、プロジェクトのルートディレクトリで以下のコマンドを実行してください。

```bash
cargo run
自動的に依存ライブラリ（macroquad, rand）がダウンロードされ、コンパイル後にゲームウィンドウが立ち上がります。
